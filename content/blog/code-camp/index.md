## Code Camp

I will be giving two presentations at the [Hampton Roads](http://www.hrssug.org/default.aspx) [Code Camp](http://www.madcodecamp.com/) in December.  The first presentation will be on OOP Exception Handling and the second on [MbUnit](http://mbunit.org/).

**Object Oriented Exception Handling**  
This session will cover extending structured exception handling into  
an OOP solution.  Reflection and polymorphism will be used to achieve  
this result.  A good understanding of polymorphism will be helpful to  
attendees.

**MbUnit**

MbUnit is an advanced .NET xUnit Framework.  This session will cover  
some of the more interesting fixtures provided by the product and a  
few custom extensions.
