## Jay Flowers

Software engineer, automation advocate, and continuous integration pioneer.

### Notable Work

- **CI Factory** -- Open-source accelerator for setting up Continuous Integration systems. Demonstrated on [dnrTV with Scott Hanselman](https://www.hanselman.com/blog/continuous-integration-screencast-jay-flowers-and-i-on-dnrtv) and [Phil Haack](https://haacked.com/archive/2007/04/29/building-a-continuous-integration-process-in-an-hour-on-dnrtv.aspx/).

- **MbUnit Contributor** -- Contributed to the MbUnit testing framework, including extensions for row-based testing and factory object patterns.

- **Windows Developer Power Tools** -- Contributor to the O'Reilly book covering essential tools for Windows developers.

- **MSDN Magazine** -- Author of ["Redefine Your Build Process with Continuous Integration"](https://learn.microsoft.com/en-us/archive/msdn-magazine/2008/march/redefine-your-build-process-with-continuous-integration) (March 2008).

### About This Site

This site is a restoration of the original jayflowers.com, which was active from 2004 through 2012. The content was recovered from the [Internet Archive's Wayback Machine](https://web.archive.org/) and converted to a static site using [Hugo](https://gohugo.io/).

The original site ran on three platforms over its lifetime:

| Era | Platform | Content |
|-----|----------|---------|
| 2004-2012 | WordPress | Blog posts on CI, testing, tools, and more |
| 2006-2008 | DokuWiki | Technical articles and pattern language documentation |
| 2006-2008 | Joomla | Downloads portal and project documentation |

Each restored post includes a link to its original Wayback Machine snapshot for provenance.
