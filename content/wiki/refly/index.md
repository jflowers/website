## Description

Refly is a code generation library, a wrapper around the .NET Framework’s CodeDom. It was originally written by Jonathan de Halleux. One of the goals of this project are to have simpler interfaces than the CodeDom, as well as cohesive. This should make for a library where the user does not need to refer to documentation to know how to use it.
